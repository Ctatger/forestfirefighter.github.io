DEMONSTRATION

Car control

Bellow, we can observe a 360-degree rotation of the car in order to detect a fire around. This action is perfomed when the car is arriving for each reached point.

website alerts

Bellow, we launched upstream the Flask server and we executed a generator of coordinates. Then, if no fire is detected we have a green icon at INSA localization either, in case of fire, we display a red icon with a photograph in the appropriate localization. By refraishing the page we see the real time reading of other localizations.

Fire detection

In this demo, the machine learning algorythm scans for fire using the camera. If a fire is detected it sends the car position using the gps sensor. The server receives the info and displays a pin on the map. A circle is also displayed to show the innacuracy of the position info
